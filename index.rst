Orbital Project Documentation
=============================

What is Orbital?
****************

Orbital is a collection of projects that are owned and maintained by
teams within Zillow Group. These projects are open sourced and can work
independently, but are also designed to work with each other.

The projects that are currently available are:

* `orbital <https://github.com/zillow/orbital>`_ : this documentation.
* `orbital-core <https://orbital-core.readthedocs.io/en/latest/>`_ : a Python library to help bootstrap aiohttp based services.
* `tycho <https://github.com/zillow/tycho>`_ : a service to track operational change
